Description

Allows you send SMS's using the smsglobal gateway
This module only sends smses, it does not yet recieve them

consider checking out v2 since it supports muliple gateways

Features

  • Configure multiple accounts for different from mobile number and credit management
  • View SMS history
  • Simple sending of smses for leads and partners
  • Allows send of smses from any model, even ones with multiple mobile number fields
  • Mass sending out to leads and partners

Using the module

Initial Configuration

1. Create a smsglobal account and add credit
2. Go to Settings->SMS Gateway Configuration->Accounts and enter the all the api settings

Sending an SMS to partners / customers / suppliers and leads

1. Simply select Send SMS from the more menu and type in your message

Sending an SMS to multiple leads / customers

1. In list view select SMS multiple partners
2. Check your sms history for smses that failed to send

Sending SMS from any model

1. Add widget="smsfield" to any field you wish treat as a mobile number
2. Click on the field to open the SMS compose window

View SMS Histroy

1. Go to Settings->SMS Gateway Configuration->SMS History
