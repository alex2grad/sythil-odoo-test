Description

Like Google Analytics but worse...
Keeps a record of http requests so you can anaylse the source of your traffic

Features

  • Keeps a record of http requests so you can anaylse the source of your traffic

Using the module

1. Put "request.env['wanal.request'].web_track(request)" in your web controller
2. View primative stats in Settings->Web Stats->Traffic Source
