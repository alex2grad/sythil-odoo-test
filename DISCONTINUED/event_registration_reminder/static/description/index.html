Description

(obsolete)Sends an email to all registrants of an event that is about to start in 2 hours

Using the module

1. Make sure your server time is correct
2. Go to Settings->Email->Templates->Event Registrants Reminder and change the template text to whatever you like.
3. Go to Settings->Automation->Scheduled Actions->Event Registrants Reminder and change how frequently you want check how frequently it checks for soon events
4. Set the action to Active and it will do the rest.

(obsolete)Can be achived with scheduled actions
