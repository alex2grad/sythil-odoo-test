Imports a large amount of customer image files(.jpg)
Designed for migrating databasing with filenames that map to database entries e.g. 576.jpg or some_restaurant.jpg
Using the module
1. Zip all the photos you wish to mass import
2. Go to Sales->Mass Import Customer Photos
3. Upload the file, set mapping field to ID if you don't understand what it's used for...
4. View MICP History for any errors that occured during the import process
5. If neccassary reupload images that failed to import after issues have been fixed
